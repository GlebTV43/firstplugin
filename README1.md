# Отчет по расширению VSCode "Tomodoro Timer"

## Краткое описание

"Tomodoro Timer" — это расширение для редактора Visual Studio Code, предоставляющее простой и наглядный помодоро-таймер прямо на панели состояния. Приложение помогает соблюдать режим работы и отдыха для повышения продуктивности.

## Основные возможности

- Рабочая сессия длительностью 50 минут.
- Перерыв 10 минут.
- Отображение периода (работа/отдых) и обратный отсчет времени в статус-баре VSCode.
- Уведомления о переключении фаз.
- Команды для управления: запуск, пауза, сброс, пропуск этапа.

## Структура проекта

- `src/extension.ts` — основной файл расширения, инициализация команд, подключение timer.
- `src/timer.ts` — реализация класса TomodoroTimer.
- `package.json` — метаданные, описание команд, зависимостей и активации.
- `package-lock.json` — фиксирует версии зависимостей.
- `CHANGELOG.md` — журнал изменений.
- `vsc-extension-quickstart.md` — краткое руководство по сборке и публикации.
- `README.md` — основное описание работы и принципов расширения.

## Главные классы и их функции

- **TomodoroTimer**
  - Управляет этапами работы и отдыха.
  - Старт, пауза, сброс.
  - Смена сессий с уведомлениями.
  - Отображение времени и статуса.

- **Extension**
  - Регистрация команд VSCode:
    - `tomodoro.start`
    - `tomodoro.pause`
    - `tomodoro.reset`
    - `tomodoro.skip`
  - Обработка событий команд.

## Использование

1. Установите расширение через Marketplace или из файла tomodoro-timer-1.0.0.vsix.
2. Запустите команду `Tomodoro: Start` через командную палитру.
3. Следуйте графику работы/отдыха.
4. При необходимости используйте команды паузы/сброса/пропуска.

## Тестирование

- Для запуска тестов используйте Extension Test Runner, инструкция приведена в vsc-extension-quickstart.md.
